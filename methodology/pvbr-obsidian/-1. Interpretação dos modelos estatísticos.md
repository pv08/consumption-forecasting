O modelo regressivo (AR model) assume que o valor corrente $y_t$ é dependente das observações passadas. Assim sendo, cria-se uma relação linear para o modelo. Aqui, será visto o modelo PACF

A média móvel (MA) assume que o valor corrente $y_t$ é dependente dos erros passados. Os erros são randomicos, então cria-se uma relação não-linear entre o valor corrente e os erros. Aqui será olhado ACF
